<h2 id="started">Getting started</h2>

**Clone the project and access the folder**
```bash
$ git clone https://github.com/marchetti2/website.git && cd website
```

**Install the dependencies**
```bash
$ yarn
```
**Run the project**
```bash
$ yarn dev
```
**Well done, project is started!**
